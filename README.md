# Gestión de Pedidos para Distribuidoras

Aplicación móvil Android orientada a optimizar la gestión de pedidos en distribuidoras y mayoristas. Reemplaza herramientas informales como Excel y WhatsApp, permitiendo registrar, consultar y analizar pedidos, clientes y productos de forma centralizada.

## Características Principales (1° Entrega)
- Arquitectura **MVVM** implementada.
- Componentes UI organizados con **Jetpack Compose**.
- Navegación gestionada a través de **Navigation Compose**.
- Internacionalización base configurada (`res/values` y `res/values-en`).
- Tema oscuro y claro implementado.
- Pantalla de Login con validaciones de ViewModel.
- Flujo interactivo en Nuevo Pedido.
- Splash Screen funcional.

## Tecnologías Utilizadas
- **Lenguaje:** Kotlin
- **UI:** Jetpack Compose (Material Design 3)
- **Navegación:** Navigation Compose
- **Arquitectura:** MVVM (ViewModel, StateFlow)
- **Datos:** Datos Mockeados en esta etapa (Preparado para Room/Retrofit en 2° Entrega).
